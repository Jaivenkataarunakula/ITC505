Explanation of HTML Tags used:

1. <!DOCTYPE html>: This declaration specifies the document type and version of HTML being used.

2. <html lang="en">: The opening <html> tag defines the root element of the HTML document. The lang attribute specifies the language of the document.

3. <head>: This section contains meta-information about the HTML document, such as character encoding, viewport settings, and title.

  • <meta charset="UTF-8">: Specifies the character encoding for the document as UTF-8.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Defines the viewport settings for responsive design on different devices.
  • <title>: Sets the title of the document displayed in the browser tab.

4. <style>: This tag contains CSS rules for styling the HTML elements within the document.

5. <body>: This section contains the content of the HTML document that is visible to the user.

6. <article>: Represents an independent piece of content within a document, such as a blog post, forum post, or news article.

7. <header>: Defines a header section typically containing introductory content, such as a title or logo.

8. <section>: Represents a thematic grouping of content, such as chapters, headings, or different parts of a webpage.

9. <h1>, <h2>, <h3>: Heading elements used to define headings of different levels (h1 being the highest level and h3 being the lowest).

10. <p>: Defines a paragraph of text.

11. <ul>, <li>: Used to create an unordered list and list items, respectively.

12. <a>: Defines a hyperlink, linking to another page or resource.

13. <footer>: Represents a footer section typically containing information about the document, such as copyright, authorship, or contact details.

14. <script>: Contains JavaScript code to add interactivity or dynamic behavior to the webpage.
